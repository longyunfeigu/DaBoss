A

Linear / Notion 工具风

紧凑、键盘优先、零装饰、信息密度最高
Open full ↗
适合：每天用 30+ 次的重度用户、技术背景、偏好 IDE 风格
  • + 信息密度最高，浏览效率最快
  • + 键盘快捷键明显，老用户友好
  • − 普通用户初见可能觉得"太冷淡"
  • − 没游戏感，难发朋友圈传播
B

DaBoss 原生游戏风

大头像、彩色卡片、CTA 突出、与现有页面一致
Open full ↗
适合：DaBoss 现有用户、追求"打怪练习"心流的用户、社交分享场景
  • + 与现有 HomePage / GrowthPage 视觉完全一致
  • + 对手画像有"角色卡"感，传播性强
  • − 信息密度低，27 条 trait 要滚很久
  • − emoji 重，国际化时文化差异风险
C

v0 / Perplexity AI 风

深色左栏 + 流式光标 + 玻璃态浮 dock，像 AI 工具
Open full ↗
适合：科技尝鲜用户、追求"被 AI 服务"心智、付费意愿高的早期采用者
  • + "AI 实时生成感"最强，体感价值最高
  • + Regenerate / Refine dock 自然支持迭代
  • − 与现有薄荷绿 + 纸感的整体不太一致，要么扩散到全站要么孤立
  • − 普通职场用户可能觉得"太极客"
D

Obsidian 学术风

双栏 + 脚注 + 引用区永久可见，证据是主角
Open full ↗
适合：高知用户、咨询/律师/HR 等"靠证据说话"的职业、严肃场景
  • + 证据链作为主角，最能建立"AI 没胡编"的信任
  • + 矛盾/反例自然展示（contradiction footnote），是其他三版做不到的
  • − 双栏布局移动端体验差，要重做
  • − serif 字体和现有 sans 系统不一致，全站风险大
