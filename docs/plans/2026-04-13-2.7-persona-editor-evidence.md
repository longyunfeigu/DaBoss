# Story 2.7 执行计划 — 5-layer Persona 编辑器 + 证据链展开

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Story**: Epic 2 / Story 2.7
**日期**: 2026-04-13
**Owner**: wanhua.gu
**依赖**: Story 2.2 (5-layer Persona 域 + repo)、Story 2.6 (`/persona/:id/edit` 路由占位 + 入口)
**阻塞**: Story 2.8 (演练对接 — 本 Story 提供 "开始演练" 按钮，2.8 接 chatroom)

**Goal**: 把 Story 2.6 跳过去的 `/persona/:id/edit` 占位页换成真正的 5-layer 编辑器：5 个 layer 卡片 + Hero + Evidence Popover + Floating CTA Bar；用户能查证据、双击编辑、标记不对、保存到 DB。

**Architecture**: 后端补 2 个 v2 端点（`GET .../personas/{id}/v2`、`PATCH .../personas/{id}/v2`）+ Persona 增 `rejected_features` 字段（存到 structured_profile JSON metadata，不动 schema_version）。前端用 Variant B 视觉令牌实现编辑器组件树：`PersonaEditorPage > [PersonaHero, LayerCard ×5, FloatingCTA]`，`LayerCard > FeatureRow ×N > EvidencePopover`。

**Tech Stack**: 后端 FastAPI + SQLAlchemy + Pydantic；前端 React 19 + TS + lucide-react + 现有 CSS 令牌。

**测试策略**:
- 后端：复用现有 pytest 模式（`backend/tests/...`），覆盖 GET v2 / PATCH v2 / repo 字段往返
- 前端：`npm run build` + `npm run lint` 必须 0 新错误；AC 末尾的 "Browser X" 手动验证
- 端到端：起后端 + 前端，用 Story 2.6 真实生成一个 persona → 点 Settings 切到列表 → 跳 `/persona/:id/edit` → 验证完整渲染

**关键设计取舍**:
- **rejected_features 存哪里**：扩展 `Persona` dataclass 加 `rejected_features: dict[str, list[int]] = {}`；在 repo 序列化时塞进 `structured_profile._metadata.rejected_features`。**不改 DB schema**（structured_profile 已是 JSONB）
- **evidence-feature 绑定方式**：每条 `Evidence.claim` 字符串与 layer 内某个特征文本完全匹配。前端构建 `Map<layer:string, Map<featureText, Evidence>>` 一次性查找。匹配不上 → popover 显示 "暂无证据，置信度未知"
- **置信度统一来源**：`Evidence.confidence`；layer 整体置信度 = 该 layer 所有 evidence 平均；Persona Strength = 全部 evidence 平均 × 100
- **PATCH 端点设计**：accept 部分 5-layer 字段（layer 缺失则保留原值）+ rejected_features 全量替换 + 不接受 evidence_citations 修改（编辑器不让用户改证据，避免破坏可追溯性）。返回 200 + 更新后 v2 payload
- **路由切换拦截**：跟 Story 2.6 一样降级 — beforeunload 覆盖刷新/关闭；SPA 内 `<Link>` 切换不拦截（声明式路由限制）。AC8 部分通过
- **"开始演练" 按钮**：本 Story 实现按钮 + click → toast 提示 "Story 2.8 will wire chatroom"，跳 Settings。Story 2.8 接管 onClick 调 `battle_prep_service.create_room_from_persona`
- **响应式 popover**：`@media (max-width: 1180px)` 时 popover 改为底部 sheet（用 fixed bottom:0 + 滑入动画）

# input: StakeholderPersonaRepository (via UoW)
# output: PersonaV2Service.get_v2 / patch_v2 — v2 编辑器后端逻辑
# owner: wanhua.gu
# pos: 应用层服务 - 5-layer Persona v2 编辑器服务 (Story 2.7)；一旦我被更新...

from __future__ import annotations
from typing import Optional
from dataclasses import asdict
from domain.stakeholder.persona_entity import (
    Persona, HardRule, IdentityProfile, ExpressionStyle, DecisionPattern, InterpersonalStyle,
)
from infrastructure.unit_of_work import UnitOfWork
from .dto import PersonaV2DTO, PersonaPatchV2DTO, EvidenceDTO


class PersonaNotFoundError(Exception):
    pass

class PersonaNotV2Error(Exception):
    pass


class PersonaV2Service:
    def __init__(self, uow_factory):
        self._uow_factory = uow_factory

    async def get_v2(self, persona_id: str) -> PersonaV2DTO:
        async with self._uow_factory() as uow:
            result = await uow.stakeholder_persona_repository.get_with_evidence(persona_id)
            if result is None:
                raise PersonaNotFoundError(persona_id)
            persona, evidence = result
            return self._to_dto(persona, evidence)

    async def patch_v2(self, persona_id: str, dto: PersonaPatchV2DTO) -> PersonaV2DTO:
        async with self._uow_factory() as uow:
            result = await uow.stakeholder_persona_repository.get_with_evidence(persona_id)
            if result is None:
                raise PersonaNotFoundError(persona_id)
            persona, evidence = result
            if persona.schema_version < 2:
                raise PersonaNotV2Error(persona_id)
            self._apply_patch(persona, dto)
            await uow.stakeholder_persona_repository.save_structured_persona(persona)
            await uow.commit()
            return self._to_dto(persona, evidence)

    def _apply_patch(self, persona: Persona, dto: PersonaPatchV2DTO) -> None:
        if dto.name is not None: persona.name = dto.name
        if dto.role is not None: persona.role = dto.role
        if dto.avatar_color is not None: persona.avatar_color = dto.avatar_color
        if dto.hard_rules is not None:
            persona.hard_rules = [HardRule(**r.model_dump()) for r in dto.hard_rules]
        if dto.identity is not None:
            persona.identity = IdentityProfile(**dto.identity.model_dump())
        if dto.expression is not None:
            persona.expression = ExpressionStyle(**dto.expression.model_dump())
        if dto.decision is not None:
            persona.decision = DecisionPattern(**dto.decision.model_dump())
        if dto.interpersonal is not None:
            persona.interpersonal = InterpersonalStyle(**dto.interpersonal.model_dump())
        if dto.rejected_features is not None:
            persona.rejected_features = dict(dto.rejected_features)

    def _to_dto(self, persona: Persona, evidence: list) -> PersonaV2DTO:
        return PersonaV2DTO(
            id=persona.id,
            name=persona.name,
            role=persona.role,
            avatar_color=persona.avatar_color,
            schema_version=persona.schema_version,
            hard_rules=[asdict(r) for r in persona.hard_rules],
            identity=asdict(persona.identity) if persona.identity else None,
            expression=asdict(persona.expression) if persona.expression else None,
            decision=asdict(persona.decision) if persona.decision else None,
            interpersonal=asdict(persona.interpersonal) if persona.interpersonal else None,
            evidence=[EvidenceDTO(**asdict(e)) for e in evidence],
            rejected_features=dict(persona.rejected_features),
        )
```

// input: persona id (路由参数) + patch payload
// output: fetchPersonaV2(id) / patchPersonaV2(id, body) — 编辑器数据访问层
// owner: wanhua.gu
// pos: 表示层 - persona v2 GET/PATCH client；一旦我被更新...

const API_BASE = '/api/v1/stakeholder'

export interface HardRule { statement: string; severity: string }
export interface Identity { background: string; core_values: string[]; hidden_agenda: string | null }
export interface Expression { tone: string; catchphrases: string[]; interruption_tendency: string }
export interface Decision { style: string; risk_tolerance: string; typical_questions: string[] }
export interface Interpersonal { authority_mode: string; triggers: string[]; emotion_states: string[] }
export interface EvidenceItem {
  claim: string
  citations: string[]
  confidence: number
  source_material_id: string
  layer: string
}
export interface PersonaV2 {
  id: string
  name: string
  role: string
  avatar_color: string | null
  schema_version: number
  hard_rules: HardRule[]
  identity: Identity | null
  expression: Expression | null
  decision: Decision | null
  interpersonal: Interpersonal | null
  evidence: EvidenceItem[]
  rejected_features: Record<string, number[]>
}
export interface PersonaPatchV2 {
  name?: string
  role?: string
  avatar_color?: string
  hard_rules?: HardRule[]
  identity?: Identity
  expression?: Expression
  decision?: Decision
  interpersonal?: Interpersonal
  rejected_features?: Record<string, number[]>
}

interface ApiResp<T> { code: number; message: string; data: T }

export async function fetchPersonaV2(id: string): Promise<PersonaV2> {
  const resp = await fetch(`${API_BASE}/personas/${id}/v2`)
  if (!resp.ok) throw new Error(`Failed to load persona: ${resp.status}`)
  const json: ApiResp<PersonaV2> = await resp.json()
  return json.data
}

export async function patchPersonaV2(id: string, body: PersonaPatchV2): Promise<PersonaV2> {
  const resp = await fetch(`${API_BASE}/personas/${id}/v2`, {
    method: 'PATCH',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(body),
  })
  if (!resp.ok) {
    const err = await resp.json().catch(() => null)
    throw new Error(err?.detail?.message || `Failed to save persona: ${resp.status}`)
  }
  const json: ApiResp<PersonaV2> = await resp.json()
  return json.data
}
```

  useEffect(() => { if (editing) inputRef.current?.focus() }, [editing])

  const startEdit = () => { if (props.onChange) setEditing(true) }
  const commit = () => {
    setEditing(false)
    if (draft !== props.text && props.onChange) props.onChange(draft)
  }
  const cancel = () => { setEditing(false); setDraft(props.text) }

  return (
    <div
      ref={rowRef}
      className={`feature-row feature ${props.rejected ? 'rejected' : ''} ${props.lowConfidence ? 'warn' : ''} ${props.evidenceFocused ? 'focused' : ''}`}
      onDoubleClick={startEdit}
    >
      <span className="emoji">{props.emoji}</span>
      <div className="text">
        {editing ? (
          <input
            ref={inputRef}
            value={draft}
            onChange={(e) => setDraft(e.target.value)}
            onBlur={commit}
            onKeyDown={(e) => {
              if (e.key === 'Enter') commit()
              else if (e.key === 'Escape') cancel()
            }}
            className="feature-edit-input"
          />
        ) : (
          <span>{props.text}</span>
        )}
      </div>
      {props.lowConfidence && <span className="warn-chip">⚠ 证据不足</span>}
      <div className="acts">
        {props.onShowEvidence && (
          <button
            ref={evidenceBtnRef}
            className={`icon-circ evidence-btn ${props.evidenceFocused ? 'focused' : ''}`}
            onClick={() => props.onShowEvidence!(rowRef.current!)}
            title="查证据"
          >
            <Search size={13} />
          </button>
        )}
        {props.onReject && (
          <button
            className="icon-circ reject-btn"
            onClick={props.onReject}
            title={props.rejected ? '取消标记' : '标记不对'}
          >
            <X size={13} />
          </button>
        )}
      </div>
    </div>
  )
}

// input: claim, citations, confidence, source_material_id, anchor (rect for positioning), onClose
// output: 320px popover (>1180px) 或 bottom sheet (≤1180px)

import { useEffect, useState } from 'react'
import { X, MessageSquare, Mail, FileText } from 'lucide-react'
import type { EvidenceItem } from '../../services/personaV2'

interface Props {
  evidence: EvidenceItem | null
  anchor: DOMRect | null
  onClose: () => void
}

const SOURCE_ICONS: Record<string, React.ReactNode> = {
  chat: <MessageSquare size={12} />,
  email: <Mail size={12} />,
  meeting: <FileText size={12} />,
  default: <FileText size={12} />,
}

function inferSourceIcon(materialId: string): React.ReactNode {
  if (materialId.includes('chat')) return SOURCE_ICONS.chat
  if (materialId.includes('mail') || materialId.includes('email')) return SOURCE_ICONS.email
  if (materialId.includes('meet') || materialId.includes('note')) return SOURCE_ICONS.meeting
  return SOURCE_ICONS.default
}

export default function EvidencePopover({ evidence, anchor, onClose }: Props) {
  const [isMobile, setIsMobile] = useState(window.innerWidth <= 1180)
  useEffect(() => {
    const onResize = () => setIsMobile(window.innerWidth <= 1180)
    window.addEventListener('resize', onResize)
    return () => window.removeEventListener('resize', onResize)
  }, [])
  useEffect(() => {
    const onKey = (e: KeyboardEvent) => { if (e.key === 'Escape') onClose() }
    window.addEventListener('keydown', onKey)
    return () => window.removeEventListener('keydown', onKey)
  }, [onClose])

  if (!evidence) return null

  // Confidence ring math
  const radius = 32
  const circumference = 2 * Math.PI * radius
  const dashOffset = circumference * (1 - Math.max(0, Math.min(1, evidence.confidence)))

  if (isMobile) {
    // bottom sheet
    return (
      <div className="evidence-sheet-overlay" onClick={onClose}>
        <div className="evidence-sheet" onClick={(e) => e.stopPropagation()}>
          <button className="sheet-close" onClick={onClose}><X size={16} /></button>
          {renderInner(evidence, radius, circumference, dashOffset)}
        </div>
      </div>
    )
  }

  // Desktop popover anchored to feature row
  const top = anchor ? anchor.top + anchor.height / 2 : 0
  const left = anchor ? anchor.right + 16 : 0
  return (
    <div
      className="evidence-popover popover"
      style={{ position: 'fixed', top, left, transform: 'translateY(-50%)', zIndex: 50 }}
    >
      {renderInner(evidence, radius, circumference, dashOffset)}
    </div>
  )
}

function renderInner(evidence: EvidenceItem, radius: number, circumference: number, dashOffset: number) {
  return (
    <>
      <div className="pop-title">{evidence.claim}</div>
      <div className="gauge">
        <svg viewBox="0 0 80 80">
          <circle cx="40" cy="40" r={radius} stroke="var(--green-soft)" strokeWidth="6" fill="none" />
          <circle
            cx="40" cy="40" r={radius}
            stroke="var(--green)" strokeWidth="6" fill="none"
            strokeDasharray={circumference}
            strokeDashoffset={dashOffset}
            strokeLinecap="round"
            transform="rotate(-90 40 40)"
          />
        </svg>
        <div>
          <div className="num">{evidence.confidence.toFixed(2)}</div>
          <div className="lbl">置信度</div>
        </div>
      </div>
      {evidence.citations.length === 0 && (
        <div className="pop-empty">暂无原文引用</div>
      )}
      {evidence.citations.map((c, i) => (
        <div key={i} className="pop-cite">
          <div className="cite-icon">{inferSourceIcon(evidence.source_material_id)}</div>
          <div className="cite-body">
            <div className="cite-time">{evidence.source_material_id}</div>
            <div className="cite-text">"{c}"</div>
          </div>
        </div>
      ))}
    </>
  )
}

// output: 完整编辑器 (Hero + 5 LayerCards + Popover + FloatingCTA)

import { useCallback, useEffect, useMemo, useState } from 'react'
import { useNavigate, useParams } from 'react-router-dom'
import { fetchPersonaV2, patchPersonaV2 } from '../services/personaV2'
import type {

  HardRule, Identity, Expression, Decision, Interpersonal,
} from '../services/personaV2'

import '../components/persona-editor/personaEditor.css'
import './PersonaEditorPage.css'

type LayerKey = 'hard_rules' | 'identity' | 'expression' | 'decision' | 'interpersonal'

const LAYER_META: Record<LayerKey, { title: string; emoji: string; color: LayerColor }> = {
  hard_rules:    { title: 'Hard Rules',    emoji: '⚖️', color: 'rose' },
  identity:      { title: 'Identity',      emoji: '🎯', color: 'violet' },
  expression:    { title: 'Expression',    emoji: '🗣️', color: 'green' },
  decision:      { title: 'Decision',      emoji: '🧠', color: 'amber' },
  interpersonal: { title: 'Interpersonal', emoji: '🤝', color: 'green' },
}

const FEATURE_EMOJI: Record<LayerKey, string> = {
  hard_rules: '⚖️', identity: '🎯', expression: '🗣️', decision: '🧠', interpersonal: '🤝',
}

const LOW_CONF = 0.6

export default function PersonaEditorPage() {
  const { id } = useParams<{ id: string }>()
  const navigate = useNavigate()
  const [persona, setPersona] = useState<PersonaV2 | null>(null)
  const [draft, setDraft] = useState<PersonaV2 | null>(null)
  const [loading, setLoading] = useState(true)
  const [error, setError] = useState<string | null>(null)
  const [saving, setSaving] = useState(false)
  const [popover, setPopover] = useState<{ ev: EvidenceItem; rect: DOMRect } | null>(null)
  const [leaveConfirm, setLeaveConfirm] = useState<null | (() => void)>(null)

  useEffect(() => {
    if (!id) return
    setLoading(true)
    fetchPersonaV2(id)
      .then((p) => { setPersona(p); setDraft(p); setError(null) })
      .catch((e) => setError(String(e)))
      .finally(() => setLoading(false))
  }, [id])

  // Build evidence map: claim text → EvidenceItem
  const evidenceMap = useMemo(() => {
    const m = new Map<string, EvidenceItem>()
    if (draft) for (const ev of draft.evidence) m.set(ev.claim, ev)
    return m
  }, [draft])

  const strength = useMemo(() => {
    if (!draft || draft.evidence.length === 0) return 0
    const sum = draft.evidence.reduce((s, e) => s + e.confidence, 0)
    return sum / draft.evidence.length
  }, [draft])

  const hasUnsaved = useMemo(() => {
    if (!persona || !draft) return false
    return JSON.stringify(persona) !== JSON.stringify(draft)
  }, [persona, draft])

  // beforeunload guard (AC8 partial)
  useEffect(() => {
    if (!hasUnsaved) return
    const handler = (e: BeforeUnloadEvent) => { e.preventDefault(); e.returnValue = '' }
    window.addEventListener('beforeunload', handler)
    return () => window.removeEventListener('beforeunload', handler)
  }, [hasUnsaved])

  const showEvidence = useCallback((claim: string, anchor: HTMLElement) => {
    const ev = evidenceMap.get(claim)
    if (!ev) return
    setPopover({ ev, rect: anchor.getBoundingClientRect() })
  }, [evidenceMap])

  const closePopover = useCallback(() => setPopover(null), [])

  const toggleReject = useCallback((layer: LayerKey, idx: number) => {
    setDraft((d) => {
      if (!d) return d
      const cur = new Set(d.rejected_features[layer] || [])
      if (cur.has(idx)) cur.delete(idx); else cur.add(idx)
      return { ...d, rejected_features: { ...d.rejected_features, [layer]: Array.from(cur).sort((a,b)=>a-b) } }
    })
  }, [])

  const isRejected = (layer: LayerKey, idx: number) =>
    Boolean(draft?.rejected_features[layer]?.includes(idx))

  const editLayerString = (layer: LayerKey, field: keyof Identity | keyof Expression | keyof Decision | keyof Interpersonal, value: string) => {
    setDraft((d) => {
      if (!d) return d
      const cur = (d as any)[layer] || {}
      return { ...d, [layer]: { ...cur, [field]: value } }
    })
  }

  const editLayerListItem = (layer: LayerKey, listField: string, idx: number, value: string) => {
    setDraft((d) => {
      if (!d) return d
      const cur = (d as any)[layer]
      if (!cur) return d
      const list = [...(cur[listField] || [])]
      list[idx] = value
      return { ...d, [layer]: { ...cur, [listField]: list } }
    })
  }

  const editHardRule = (idx: number, value: string) => {
    setDraft((d) => {
      if (!d) return d
      const list = [...d.hard_rules]
      list[idx] = { ...list[idx], statement: value }
      return { ...d, hard_rules: list }
    })
  }

  const handleSave = async () => {
    if (!draft || !id) return
    setSaving(true)
    try {
      const body: PersonaPatchV2 = {
        name: draft.name,
        role: draft.role,
        avatar_color: draft.avatar_color || undefined,
        hard_rules: draft.hard_rules,
        identity: draft.identity || undefined,
        expression: draft.expression || undefined,
        decision: draft.decision || undefined,
        interpersonal: draft.interpersonal || undefined,
        rejected_features: draft.rejected_features,
      }
      const updated = await patchPersonaV2(id, body)
      setPersona(updated)
      setDraft(updated)
    } catch (e: any) {
      setError(e?.message || String(e))
    } finally {
      setSaving(false)
    }
  }

  const handleStartBattle = () => {
    // Story 2.8 will wire this — for now, navigate to chat list with toast
    alert('演练接入即将在 Story 2.8 上线')
    navigate('/chat')
  }

  const guardedNav = (target: string) => {
    if (hasUnsaved) setLeaveConfirm(() => () => navigate(target))
    else navigate(target)
  }

  if (loading) return <div className="editor-loading">加载中…</div>
  if (error || !draft) return (
    <div className="editor-error">
      加载失败：{error || 'persona 不存在'}
      <button onClick={() => navigate('/settings')}>返回 Settings</button>
    </div>
  )

  // -------- 渲染 5-layer 列表 --------

  const hardRulesRows = draft.hard_rules.map((r, idx) => {
    const ev = evidenceMap.get(r.statement)
    return (
      <FeatureRow
        key={idx}
        emoji={FEATURE_EMOJI.hard_rules}
        text={r.statement}
        rejected={isRejected('hard_rules', idx)}
        lowConfidence={ev ? ev.confidence < LOW_CONF : false}
        evidenceFocused={popover?.ev.claim === r.statement}
        onChange={(v) => editHardRule(idx, v)}
        onShowEvidence={ev ? (anchor) => showEvidence(r.statement, anchor) : undefined}
        onReject={() => toggleReject('hard_rules', idx)}
      />
    )
  })

  // Identity: scalar fields → 3 rows
  const identityRows: React.ReactNode[] = []
  if (draft.identity) {
    const { background, core_values, hidden_agenda } = draft.identity
    if (background) identityRows.push(
      <FeatureRow key="bg" emoji={FEATURE_EMOJI.identity} text={background}
        onChange={(v) => editLayerString('identity', 'background', v)} />
    )
    core_values.forEach((cv, i) => identityRows.push(
      <FeatureRow key={`cv-${i}`} emoji="✦" text={cv}
        onChange={(v) => editLayerListItem('identity', 'core_values', i, v)} />
    ))
    if (hidden_agenda) identityRows.push(
      <FeatureRow key="ha" emoji="🕶️" text={`隐藏议程：${hidden_agenda}`}
        onChange={(v) => editLayerString('identity', 'hidden_agenda', v.replace(/^隐藏议程：/, ''))} />
    )
  }

  // Expression
  const exprRows: React.ReactNode[] = []
  if (draft.expression) {
    const { tone, catchphrases, interruption_tendency } = draft.expression
    if (tone) exprRows.push(<FeatureRow key="tone" emoji={FEATURE_EMOJI.expression} text={`语气：${tone}`}
      onChange={(v) => editLayerString('expression', 'tone', v.replace(/^语气：/, ''))} />)
    catchphrases.forEach((p, i) => exprRows.push(

    ))
    exprRows.push(<FeatureRow key="int" emoji="✋" text={`打断倾向：${interruption_tendency}`}

  }

    const { authority_mode, triggers, emotion_states } = draft.interpersonal
    if (authority_mode) ipRows.push(<FeatureRow key="am" emoji={FEATURE_EMOJI.interpersonal} text={`权威模式：${authority_mode}`}
      onChange={(v) => editLayerString('interpersonal', 'authority_mode', v.replace(/^权威模式：/, ''))} />)
    triggers.forEach((t, i) => ipRows.push(
      <FeatureRow key={`tr-${i}`} emoji="⚡" text={`触发器：${t}`}
        onChange={(v) => editLayerListItem('interpersonal', 'triggers', i, v.replace(/^触发器：/, ''))} />
    ))
    emotion_states.forEach((s, i) => ipRows.push(
      <FeatureRow key={`es-${i}`} emoji="😤" text={s}
        onChange={(v) => editLayerListItem('interpersonal', 'emotion_states', i, v)} />

  }

  return (
    <div className="persona-editor">
      <PersonaHero
        persona={draft}
        strength={strength}
        evidenceCount={draft.evidence.length}
        materialCount={0 /* source_materials not in v2 dto yet */}

      <LayerCard title={LAYER_META.hard_rules.title} emoji={LAYER_META.hard_rules.emoji}
        color={LAYER_META.hard_rules.color} count={hardRulesRows.length}>

}
